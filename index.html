Monthly Feature

Every month I update this section to give an idea of what I'm either working on or learning!

Here is an interactive list that I created using html, css, javascript, and jquery. I used handlebars js to create the template for the list items. The list items are created from the input box submission. To add extra functionality I included checking a task as done by clicking, removing individual items, auto sorting completed to the bottom of the list, and having a clear all button.
