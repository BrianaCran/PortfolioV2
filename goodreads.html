Goodreads

Goodreads is an online website that promotes interacting with authors and other book readers. One of the biggest features of the site is the abillity to add books to lists of currently reading, have read, and to read. These options make up the bookself page which was in need of a redesign.

Process

Goodreads already has a large user base and so the redesign would need to stay within branding, but provided improved usability and easy navigation.
